API Reference#

Input/Output#

from_file(file_path, source_software[, fps])

Load pose tracking data from a DeepLabCut (DLC), LightningPose (LP) or SLEAP output file into an xarray Dataset.

from_sleap_file(file_path[, fps])

Load pose tracking data from a SLEAP file into an xarray Dataset.

from_dlc_file(file_path[, fps])

Load pose tracking data from a DeepLabCut (DLC) output file into an xarray Dataset.

from_dlc_df(df[, fps])

Create an xarray.Dataset from a DeepLabCut-style pandas DataFrame.

from_lp_file(file_path[, fps])

Load pose tracking data from a LightningPose (LP) output file into an xarray Dataset.

to_dlc_file(ds, file_path[, split_individuals])

Save the xarray dataset containing pose tracks to a DeepLabCut-style .h5 or .csv file.

to_dlc_df(ds[, split_individuals])

Convert an xarray dataset containing pose tracks into a single DeepLabCut-style pandas DataFrame or a dictionary of DataFrames per individual, depending on the 'split_individuals' argument.

to_sleap_analysis_file(ds, file_path)

Save the xarray dataset containing pose tracks to a SLEAP-style .h5 analysis file.

to_lp_file(ds, file_path)

Save the xarray dataset containing pose tracks to a LightningPose-style .csv file.

ValidFile(path, *[, expected_permission, ...])

Class for validating file paths.

ValidHDF5(path, *[, expected_datasets])

Class for validating HDF5 files.

ValidPosesCSV(path)

Class for validating CSV files that contain pose estimation outputs.

ValidPoseTracks(*, tracks_array[, ...])

Class for validating pose tracking data imported from a file.

report_nan_values(ds[, ds_label])

Report the number and percentage of points that are NaN for each individual and each keypoint in the provided dataset.
